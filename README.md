# josephlbautista.com

Personal portfolio site for Joseph Lorenzo Bautista — CS student at CSU Chico.

Built with Next.js, TypeScript, and Tailwind CSS. Features a 3D cat model rendered with Three.js and a minimal monospace aesthetic.

## Pages

- **Home** — bio, contact links, projects, and skills
- **Experiences** — education and work history

## Stack

- [Next.js](https://nextjs.org) (App Router)
- TypeScript
- Tailwind CSS
- Three.js / React Three Fiber (3D cat scene)

## Running locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploying

Deployed on [Vercel](https://vercel.com). Pushes to `main` trigger automatic redeployment.
